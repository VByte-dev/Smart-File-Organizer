# Smart File Organiser

Smart File Organiser is a Python script that automatically organises files in a directory based on their file extensions.

## Features
- Supports organisation of various file types into specific folders.
- Creates folders for common file types such as documents, images, videos, music, and Python files.
- Handles errors gracefully and provides informative messages.

## Usage
1. Run the script and provide the path of the directory you want to organise.
2. The script will identify the file types based on their extensions and move them to corresponding folders.
3. Files with unsupported extensions will remain in the original directory.
4. Once the organisation is complete, a success message will be displayed.

## Supported File Types
- Documents: .doc, .docx, .txt, .pdf
- Images: .jpg, .jpeg, .png, .gif
- Videos: .mp4, .avi, .mov
- Music: .mp3, .wav
- Python Files: .py

## Requirements
- Python 3.x
- shutil module (included in the Python standard library)
